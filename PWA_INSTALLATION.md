# Guia de Instalação do PWA no iPhone

## O que é um PWA?

Um **Progressive Web App (PWA)** é um aplicativo web que funciona como um app nativo no seu iPhone, mesmo sem estar na App Store. Ele pode ser instalado na home screen e funciona offline!

## Como Instalar no iPhone

### Passo 1: Abrir no Safari
1. Abra o Safari no seu iPhone
2. Navegue para: `https://hidratacao-gfsrycf4.manus.space` (ou o domínio do seu app)

### Passo 2: Adicionar à Tela de Início
1. Toque no ícone de **Compartilhar** (caixa com seta para cima)
2. Role para baixo e toque em **"Adicionar à Tela de Início"**
3. Personalize o nome (padrão: "Meu Cantinho da Hidratação")
4. Toque em **"Adicionar"**

### Passo 3: Pronto!
- O app aparecerá na sua home screen com o ícone rosa
- Toque para abrir e usar como um app normal
- Funciona offline após a primeira visita!

## Funcionalidades do PWA

✅ **Instalação na Home Screen** - Sem precisar da App Store
✅ **Modo Fullscreen** - Interface sem barra do navegador
✅ **Funcionamento Offline** - Acessa dados em cache quando sem internet
✅ **Sincronização** - Salva dados quando volta online
✅ **Notificações** - Recebe alertas do app
✅ **Acesso Rápido** - Atalho para registrar hidratação

## Dicas

- **Primeira Visita**: Abra o app no navegador uma vez para fazer cache dos arquivos
- **Offline**: O app funciona offline, mas precisa de internet para sincronizar dados
- **Atualizar**: O app se atualiza automaticamente quando há novas versões
- **Remover**: Pressione e segure o ícone, depois toque em "Remover"

## Requisitos

- iPhone com iOS 11.3 ou superior
- Safari (não funciona em Chrome/Firefox no iOS)
- Conexão com internet para primeira instalação

## Solução de Problemas

### "Adicionar à Tela de Início" não aparece
- Certifique-se de estar usando Safari
- Tente recarregar a página
- Verifique se o iOS está atualizado

### App não funciona offline
- Abra o app no navegador uma vez para fazer cache
- Espere alguns segundos para o Service Worker se registrar
- Tente desligar e ligar o WiFi

### App não atualiza
- Feche e reabra o app
- Limpe o cache do Safari: Configurações > Safari > Limpar Histórico e Dados
- Reinstale o app
